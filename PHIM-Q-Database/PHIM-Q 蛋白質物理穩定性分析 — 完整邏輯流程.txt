## PHIM-Q 蛋白質物理穩定性分析 — 完整邏輯流程

---

### 🎯 一句話總結

**PHIM-Q 是從蛋白質序列出發，透過雙軌物理模型計算「病理指數 (PI)」，評估蛋白質是否容易失穩、聚集，並可偵測隱藏致病核心的工具。**

## 🔬 各物理參數的意義

| 參數 | 全名 | 物理意義 |
|------|------|----------|
| **HC** | Hydrophobic Cluster | 疏水胺基酸在序列中的聚集程度。越高代表蛋白質容易形成疏水核心，但也可能暴露而聚集。 |
| **Beta** | Beta Propensity | 形成 β-摺疊結構的傾向。β-摺疊是澱粉樣纖維的核心結構。 |
| **Disorder** | Disorder Tendency | 無序區比例。IDP 的聚集機制與疏水蛋白不同，需要獨立處理。 |
| **Psi** | Charge Heterogeneity | 電荷在序列中的不均勻分布程度。電荷失衡會破壞靜電屏蔽。 |
| **Xi_W** | Charge Perturbation | 電荷擾動強度，反映靜電護盾的穩定性。 |
| **W_IDP** | IDP Weight | 蛋白傾向 IDP 軌道的權重。由 Disorder 和 Beta 共同決定。 |
| **Λ(L)** | Length Factor | 長度因子。長蛋白 (IDP) 權重增加，長摺疊蛋白權重降低。 |
| **Baseline** | Reference Baseline | 結構穩定蛋白的 Omega 中位數 (0.3)。用於標準化 PI。 |

## 🧠 核心邏輯背後的物理直覺

### 1️⃣ 為什麼要分「摺疊軌道」和「IDP 軌道」？

```
摺疊蛋白 (Folded)         無序蛋白 (IDP)
────────────────────────────────────────────────
有固定 3D 結構             無固定結構
疏水核心在內部             疏水性低
聚集 = 疏水暴露           聚集 = 電荷崩潰 + 無序區塌陷
藥物策略 = 穩定結構       藥物策略 = 恢復靜電屏蔽
```

**兩種蛋白的「物理弱點」完全不同，必須分開處理。**

---

### 2️⃣ 為什麼要「核心掃描」？

```
全長 APP (770 aa)          Aβ42 片段 (42 aa)
────────────────────────────────────────────────
PI = 0.89 (穩定)           PI = 1.56 (高風險)

全長蛋白把致病核心「埋」在裡面，平均化後看不出風險。
核心掃描 = 找出這些隱藏的「定時炸彈」。
```

---

### 3️⃣ 為什麼要用「對數混合」？

```
Omega_F 和 Omega_I 的數量級可能差異很大（例如 0.1 vs 1.5）
直接線性混合會讓大的一方主導結果。

log1p 壓縮後，兩者權重更平衡。
```

---

### 4️⃣ 為什麼「分段縮放」？

```
PI_Raw 可能低至 0.5、高至 3.0+
直接輸出會讓低風險區壓縮、高風險區不明顯。

分段縮放：低區線性、中區輕壓縮、高區對數壓縮
→ 保留高風險區的解析度。
```

## 📈 PI 的意義與應用

| PI 範圍 | 風險 | 意義 |
|---------|------|------|
| < 0.8 | 極低 | 非常穩定，不易聚集 |
| 0.8 - 1.0 | 低 | 正常蛋白，如對照組 |
| 1.0 - 1.2 | 中 | 邊界風險，需注意 |
| 1.2 - 1.5 | 中高 | 可能容易聚集 |
| > 1.5 | 高 | 高風險，易失穩/聚集 |

## ✅ 總結一句話

> **PHIM-Q 用物理學的角度，把蛋白質的「穩定度」變成一個可以計算、可以比較、可以篩選的數字 (PI)。它不是告訴你蛋白質長什麼樣，而是告訴你它「會不會壞掉」。**
